# Steam Badge Helper

Tampermonkey 用户脚本，在 Steam 徽章页面扫描有未完成进度的游戏徽章，批量查询卡牌价格，计算Steam市场购买卡牌套装价格。

> **当前版本 v0.8.0**：购买功能尚未实现，版本号未到 1.0 正是因此。目前仅支持扫描与价格估算。

## 功能

- 自动扫描徽章页面，找出有未完成进度的游戏
- 批量查询卡牌市场价格（含最低价与中位数）
- 按单套卡牌价格上限筛选
- 满级价格估算（最低价 + 中位数混合）

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 导入 `steam-badge-helper.user.js`
3. 打开任意 Steam 徽章页 (如 `steamcommunity.com/id/xxx/badges/`)
4. 点击页面右侧 "徽章卡牌购买助手" 按钮

## 使用说明

1. 点击入口按钮打开助手面板
2. 设置 **单套卡牌价格上限**（人民币）
3. 点击 **开始扫描**，等待扫描完成
4. 勾选感兴趣的游戏，查看价格估算，手动购买

## 配置项

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 单套卡牌价格上限 | ¥10 | 全套卡牌最低价之和的上限 |
| 最大徽章页数 | 10 | 扫描的最大页数 |
| 包含掉落 | 关 | 是否包含仍有掉落的游戏 |
| 请求间隔 | 1200ms | 每次请求间隔 |
| 批暂停 | 20次/15s | 防止限流 |

## 注意事项

- 仅支持人民币 (CNY) 交易
- 满级价格为估算值，成交量低时可能不准
- 不支持闪卡
